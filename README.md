#blade-bbs

使用java搭建的简洁论坛系统，基于 [Blade](https://github.com/biezhi/blade) 框架。

## 演示站点

[http://139.196.5.229](http://139.196.5.229)

## 特性

+ 支持markdown语法
+ 支持移动端访问
+ 支持AT功能
+ 多主题支持
+ 高效快速相应
+ Java8语法
